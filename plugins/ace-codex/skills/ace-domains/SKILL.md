---
name: ace-domains
description: Use when the user wants to list the pattern domains available in the ACE playbook for use as `--allowed-domains` or `--blocked-domains` filters in search and retrieval.
---

Invoke directly with `$ace-domains`, or ask `@ace-codex` for domain filtering options.

Run `ace-cli domains`. Useful flags:
- `--min-patterns <n>` to hide domains with fewer patterns (default 1)
- `--json` for machine output
