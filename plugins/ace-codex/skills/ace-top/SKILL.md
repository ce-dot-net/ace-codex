---
name: ace-top
description: Use when the user wants to see the highest-rated ACE patterns by helpful score, optionally limited or filtered by section.
---

Invoke directly with `$ace-top`, or ask `@ace-codex` for top-rated patterns.

Run `ace-cli top`. Useful flags:
- `--limit <n>` to override the server adaptive default
- `--section <strategies|snippets|troubleshooting|apis>` to filter by section
- `--json` for machine output
